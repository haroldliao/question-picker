# 活動提問轉盤

這是一個可直接用 GitHub Pages 發布的單頁活動提問網頁。

## 使用方式

開啟 `index.html` 即可使用。

- `出題`：隨機抽出尚未抽過的題目
- `Reset`：清空抽題紀錄並重新排列題目

## GitHub Pages 發布

1. 在 GitHub 建立新的 repository。
2. 將此資料夾推送到該 repository。
3. 到 repository 的 `Settings` > `Pages`。
4. `Source` 選擇 `Deploy from a branch`。
5. `Branch` 選擇 `main`，資料夾選擇 `/root`。
6. 儲存後等待 GitHub Pages 產生公開網址。

發布完成後，可用 Pad 直接開啟該網址，或將網址製作成 QR Code。
